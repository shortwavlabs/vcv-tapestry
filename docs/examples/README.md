# Tapestry Examples

This directory contains real-world implementation examples demonstrating various use cases for Tapestry.

---

## Example Categories

### 1. [Basic Recording and Playback](01_basic_recording.md)
Simple recording, playback, and splice creation workflows.

### 2. [Granular Textures](02_granular_textures.md)
Creating ambient textures, pads, and atmospheric soundscapes.

### 3. [Rhythmic Processing](03_rhythmic_processing.md)
Clock-synced granulation for drums, percussion, and rhythmic material.

### 4. [Vocal Processing](04_vocal_processing.md)
Granular vocal effects, harmonization, and manipulation.

### 5. [Live Performance](05_live_performance.md)
Real-time looping, live manipulation, and performance setups.

### 6. [Sound Design](06_sound_design.md)
Creating unique textures, evolving sounds, and experimental effects.

### 7. [Expander Effects](07_expander_effects.md)
Using the Tapestry Expander for bit crushing and filtering.

### 8. [Integration Patterns](08_integration_patterns.md)
Combining Tapestry with other VCV Rack modules for complex patches.

---

## How to Use These Examples

Each example includes:
- **Goal**: What the example demonstrates
- **Required Modules**: List of VCV Rack modules needed
- **Patch Diagram**: ASCII or description of connections
- **Step-by-Step Instructions**: Detailed setup guide
- **Parameter Settings**: Recommended starting values
- **Tips and Variations**: Ways to modify and explore
